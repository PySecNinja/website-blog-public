A modern, lightweight blog built with FastAPI and Jinja2 templates.

## Features

- **Markdown Content** - Write posts and projects in Markdown with YAML frontmatter
- **Admin Dashboard** - Simple password-protected admin interface
- **Responsive Design** - Clean, mobile-friendly layout
- **Syntax Highlighting** - Code blocks with Pygments highlighting

## Tech Stack

- FastAPI for the backend
- Jinja2 for templating
- python-markdown for content parsing
- Pure CSS (no frameworks)

## Getting Started

```bash
pip install -r requirements.txt
python main.py
```

Visit `http://localhost:8000` to view the site.
